# SF6-ranked-match-analysis
My first data analysis project revolving around analysing my own ranked match data to see what skills translate into winning a match. This project is an attempt to apply what I learned from IBM and Google's Data Analytics courses offered on Coursera and combine it with my hobby of playing Street Fighter 6. I was curious to see if I could use what I learned in my courses to see if certain patterns in my play could help predict whether a match would result in a win or loss.

For simplicity's sake, I wanted to see if Anti-airs, combo drops, burnouts and opponents MR were strong predictors for determining whether I won or lost a match. Full descriptions of each feature is explained in within the notebook.

The tools used for this project was Python utilizing pandas, matplotlib, seaborn, and sklearn.
